# Business Sales Analysis Using Excel


## Project Overview 

This project analyses business sales data using Microsoft Excel. The project aims to summarize, visualize and interpret sales performance to provide insights for business, such as trends and possible improvement areas. 

Moreover the project showcases excel-based analysis skills such as data cleaning, formulas, PivotTables, PivotCharts, dashboard design and business writing.

# Project Process

1. Import the raw sales dataset.
2. Clean the data using Excel and/or Power Query.
3. Check for missing values, duplicates, and inconsistent formatting.
4. Create calculated fields if needed. 
5. Build PivotTables to summarize sales performance. 
6. Create charts to visualize trends and comparison.
7. Design a simple Excel Dashboard.
8. Interpret the results and write business insights.
9. Provide recommendations based on the analysis.
